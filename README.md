# plenv plugin install

Install [plenv](https://github.com/tokuhirom/plenv) plugins.

# Install

```
❯ git clone https://github.com/skaji/plenv-plugin-install $(plenv root)/plugins/plenv-plugin-install
```

# Usage

```
❯ plenv plugin-install download
```
